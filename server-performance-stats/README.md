## 🧪 ¿Qué hace este script?

- Muestra el **top 5 de procesos que más CPU consumen**
- Muestra el **top 5 de procesos que más RAM consumen**
- Muestra el **uso total del disco** (espacio total, usado, libre y porcentaje)
